How to run the Broadband Speed Test

Click the "Start Speed Test" button on the broadband speed test below and then the speed test will automatically run and figure out what your broadband speed is for both download and upload.

Make sure your internet connection is not being used for anything else while the broadband speed test runs the speed test (takes less than half a minute) so that we can give you an accurate result!
