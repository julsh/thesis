Search Engines and Directories
Internet search engines help users find web pages on a given subject. The
search engines maintain databases of web sites and use programs (often
referred to as "spiders" or "robots") to collect information, which is then
indexed by the search engine. Similar services are provided by "directories"
which maintain ordered lists of websites.

For best results using search engines in other languages we
recommend using Chrome with Google automatic translation.
