Web Statistics and Analytics
Web analytics is the measurement of the behaviour of visitors to a website.
In a commercial context, it especially refers to the measurement of
which aspects of the website work towards the business objectives.
Several different vendors provide web analytics software and services.

››› RANKING WEB PAGES AND TRAFFIC

Google Page Rankings Tool Bar
A must-have tool for search engine marketers,
website optimizers, and pay-per-click advertisers.
Includes free pop-up manager and Page Rank.

 Alexa Traffic Rankings
Alexa is a very powerful tool used to rank web site traffic. The
tool bar includes free pop-up manager and Alexa Traffic Rank.


 A9 Search Engine
A very special search engine that is powered
by Google, belongs to Amazon, and links to
Alexa Traffic Rank for each site searched.

 IAB - Interactive Advertising Bureau
The IAB is an association dedicated to helping online,
Interactive broadcasting, email, wireless and Interactive
television media companies increase their revenues.

 Compete Search Engine
A different style of search engine that helps you find sites and also
shows you how safe, popular and valuable each site is. Compete.com
extends the search algorithms, creating useful new information that can
help answer questions like: Is this website safe from spyware and other
threats like phishing? How many people visit this site and how does it compare
to other sites? Are there promotion codes for this site that can save me money?
An interesting concept based on click sharing. Compete is powered by Yahoo!

 Quantcast Internet Ratings
Quantcast is the world's first open internet ratings service.
Advertisers can find reports on the audiences of millions of web sites.
Publishers can ensure their sites are represented accurately by
tagging them for direct measurement. The service is free.

 Yahoo Site Explorer
Use Yahoo! Site Explorer to measure webpage count and inbound links, a free service from Yahoo, currently in Beta, that provides indexed website page counts at the domain and site-wide basis, and links from other web pages count, both internal and external.
